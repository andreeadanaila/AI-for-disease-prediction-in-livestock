Frequently Asked Questions (FAQ)

What is this AI system for?
This AI system helps predict livestock diseases based on symptoms you observe. It suggests possible conditions and provides details like severity, duration, curability, and affected animal types.
How accurate are the predictions?
Predictions are generated using a real-world dataset of known diseases and symptoms. While the system offers helpful insights, it should not replace consultation with a licensed veterinarian.
How do I use the system?
Select the type of livestock and mark the symptoms shown. You can also describe additional symptoms in a text box. Once submitted, the system predicts the likely disease and displays relevant information.
Can I use this system for any type of livestock?
Yes. The system supports multiple livestock types including cattle, goats, sheep, pigs, poultry, and horses. Results are specific to the selected animal type.
What if my animal has symptoms that aren’t listed?
You can manually describe unlisted symptoms in the "Other symptoms" field. The system will still use the provided data to make the best possible prediction.
How can I be sure the results are reliable?
The system uses machine learning trained on real disease-symptom relationships. It is intended as a decision-support tool, not a substitute for professional veterinary advice.
Can I trust the system to provide treatment recommendations?
No. While it may provide curability information, it does not give specific treatments. Always consult a veterinarian for appropriate care plans.

Is the data I enter kept private?
Yes. Any data you submit is used solely for prediction purposes and is never shared with third parties. Your privacy is important to us.
