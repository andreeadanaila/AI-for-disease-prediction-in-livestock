About

This project aims to provide an intelligent system for predicting livestock diseases based on common symptoms. By leveraging machine learning algorithms and a comprehensive dataset of diseases, symptoms, and affected animals, the system assists farmers and veterinarians in early disease detection, helping improve animal welfare and productivity.

The system works by allowing users to input symptoms they observe in their animals and receive a disease prediction based on those symptoms. This prediction is supported by detailed information about the disease, including its severity, affected animals, duration, and curability.

Key Features

  • Disease Prediction: Using a set of symptoms provided by the user, the AI model predicts the most likely disease.
  • Animal-specific Information: Results include the affected animals, providing insights specific to different livestock species.
  • Data Transparency: The system offers a detailed report, including disease symptoms, severity, and curability, based on a real-world dataset of diseases.

By providing quick, data-driven predictions, this project aims to reduce diagnostic time, improve early detection, and ultimately support better management of livestock health.

Technologies Used

  • Python & Pandas: For data processing and building the dataset of diseases and symptoms.
  • AI Model (Random Forest): This is used to predict the disease based on the symptoms provided.
  • Django: This is for creating the backend system and handling data requests and predictions.
  • HTML, CSS, JavaScript: For building a user-friendly web interface.
